name: React Router - User profile page
---

## story

As a user, I want each directory entry to have its own profile route,
so that I can inspect contact details and return to the user list.

## acceptance criteria

- [ ] When I open the Users page and click the first visible "View Profile" button, then the URL matches `/users/:id`.
- [ ] When the user profile page loads, then I see the selected user's name and email together with a link back to Users.
- [ ] When the user profile page loads, then I see the sections "Contact Information" and "Role & Department".
- [ ] When the user profile page loads, then I see fields for email, phone, location, join date, role, and department.
- [ ] When I use the back link above the profile, then I return to the Users list route.
