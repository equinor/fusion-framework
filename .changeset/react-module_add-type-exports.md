Add type exports for better TypeScript support and tree-shaking.

The following types are now exported from `@equinor/fusion-framework-react-module`:
- `Modules`
- `ModulesInstanceType`
- `AnyModule`
- `ModulesInstance`

```typescript
import type { Modules, ModulesInstanceType } from '@equinor/fusion-framework-react-module';
```
